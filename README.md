# Movie Recommender System

A content-based movie recommendation system built using **Python**, **Pandas**, **Scikit-Learn**, and **Streamlit**. The application recommends movies similar to a selected movie based on movie metadata and similarity scores.

## Features

* Search and select movies from a dropdown menu.
* Get personalized movie recommendations instantly.
* Uses content-based filtering for recommendations.
* Interactive web interface built with Streamlit.
* Fast recommendation generation using precomputed similarity matrices.

## Dataset

The project uses the TMDB 5000 Movie Dataset, which contains information such as:

* Movie titles
* Genres
* Keywords
* Cast
* Crew
* Overview

These features are processed and combined to create a representation of each movie.

## How It Works

1. Movie metadata is cleaned and preprocessed.
2. Relevant textual features are combined into a single feature vector.
3. Text data is transformed using vectorization techniques.
4. Cosine similarity is computed between movies.
5. When a user selects a movie, the system finds the most similar movies and displays recommendations.

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd movie-recommender-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Running the Application

Start the Streamlit application:

```bash
streamlit run app.py
```

Open the local URL generated by Streamlit in your browser.

## Example

1. Select a movie from the dropdown.
2. Click **Show Recommendation**.
3. View the recommended movies generated by the similarity model.
