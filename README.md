# 🎬 Movie Recommender System

[![Live Demo](https://img.shields.io/badge/Live-Demo-FF4B4B?style=for-the-badge&logo=streamlit&logoColor=white)](https://movierecommendsystm.streamlit.app/)
[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github)](https://github.com/GoodBoy20/Movie_Recommender_System)

A **content-based Movie Recommendation System** built using **Python**, **Pandas**, **Scikit-Learn**, and **Streamlit**. The application recommends movies similar to a selected movie by analyzing movie metadata and computing cosine similarity between feature vectors.

# ✨ Features

- 🎥 Search and select movies from an interactive dropdown.
- 🤖 Get instant content-based movie recommendations.
- 📊 Uses cosine similarity for accurate recommendations.
- ⚡ Fast recommendation generation using precomputed similarity matrices.
- 💻 Interactive web interface built with Streamlit.

---

# 📚 Dataset

This project uses the **TMDB 5000 Movie Dataset**, which includes:

- Movie Titles
- Genres
- Keywords
- Cast
- Crew
- Overview

These features are preprocessed and combined into a single feature representation used to compute movie similarity.

Dataset:
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata

---

# ⚙️ How It Works

1. Load and preprocess movie metadata.
2. Combine important textual features into a single **tags** column.
3. Convert text into numerical vectors using **CountVectorizer**.
4. Compute cosine similarity between all movies.
5. Recommend the top five most similar movies based on the selected movie.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/GoodBoy20/Movie_Recommender_System.git
cd Movie_Recommender_System
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```

Then open the local URL generated by Streamlit in your browser.

---

# 🎯 Usage

1. Select a movie from the dropdown.
2. Click **Show Recommendation**.
3. View the top five recommended movies instantly.
